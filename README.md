# Web Technology Projects

This repository contains a web-based project built using core web technologies for interactive and dynamic user experiences.

---

## Projects

### Quiz System

A Quiz System developed using Web Technologies for creating, managing, and attempting quizzes with real-time scoring and result tracking.

**Features**
- Start and attempt quiz
- Multiple choice questions (MCQ)
- Timer for each question
- Real-time score tracking
- View result and correct answers
- Admin: Add / Edit / Delete questions

---

## Technology Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage (for data persistence)
